Where AI must stop
At the Execution Boundary. Under IIROC/CSA (CIRO) regulations, personalized rebalancing requires a suitability confirmation. Beyond compliance, this is a vital trust-gate. AI handles the discovery and the math, but the final "Execute" command must remain human. This ensures the system acts as a co-pilot, not an autonomous agent that could misinterpret a user's changing life circumstances (e.g., a sudden need for liquidity) that wasn't captured in the chat.

What would break first at scale
The Alert Infrastructure (The "Flash Mob" Problem). In a macro event—like an OPEC announcement—the system would simultaneously trigger thousands of "Watchdog" alerts for users holding energy tickers (ENB, IXC). This creates a spike in "intent" that could overwhelm the human advisor pipeline for confirmations.

Exposure: Users with the highest concentration risk.

Timing: Users whose monthly DCA deposit is scheduled within the next 48 hours.

Sentiment: Prioritizing users who indicated "High Anxiety" during their Socratic chat history.
